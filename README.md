# bookmarks-checker
>Счётчик закладок в подпапках каталога Firefox.
## Описание
    Скрипт анализирует указанную в конфигурации директорию, подсчитывая количество закладок и вложенных папок. Аналогичная проверка выполняется и для следующего уровня вложенности, а итоговый отчет сохраняется в файл docs/Result.txt.
## Запуск
### Активация скрипта из папки проекта
    python src/start.py
